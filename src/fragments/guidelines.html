Naming

Tokens: category/role-variant (e.g. color/accent-soft). Components: PascalCase. Files & branches: kebab-case.

How to apply the system day to day.

Reuse vs. create

Start from an existing component. If it's 80% there, extend it with a variant. Create a new component only when the pattern is genuinely new and will be reused.

Exceptions

Edge cases happen. When you must deviate, document why in the pull request so the decision is visible — an undocumented exception is a bug.
